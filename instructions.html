Instructions for CHS - Period Notifier:

The Period Notifier is a web application that displays system date, time, period, period duration and type of day:

  • Date: system date
  • Time: system time
  • Period: Period 0, Period 1-8, Afterschool
  • Period duration: school starts countdown, period duration
  • Notes:
    • On Regular Schedule = Follows regular schedule
    • Homeroom Day = Appears on 15th and end of month
    • Weekend = Appears on weekends

Top bar:
  • Displays title of the web app
  • Period and period duration appears when on another page
  • When on another page, press it to go back to main page

Bottom bar:
  • School shaped: Link to chs home page
    • Press top bar to return to home, DONT PRESS BACK BUTTON
  • Calendar shaped: Link to chs calendar
    • Press top bar to return to home, DONT PRESS BACK BUTTON
  • Clock shaped: Link to full schedule; changes if homeroom day
    • Opens dialogue box
  • Face shaped: Link to credits and author notes
    • Opens dialogue box
    • School time difference (must be manually changed internally)
  • Question shaped: Link to instructions (this)
    • Opens dialogue box

Orientation:
  • Portrait/Vertical: Top/bottom boxes appear all the time
  • Landscape/Horizontal: Top/bottom boxes appear sometimes
    • Disappears with certain smaller height screen resolutions (some small phones)

Background:
  • Dynamically changes:
    • Before school: Brightened CHS image
    • During school: Normal CHS image
    • After school: Darkened CHS image
